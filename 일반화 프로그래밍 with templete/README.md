## 1. 일반화 프로그래밍
1) 함수 템플릿
   * 범용성 있는 자료형 1개를 매개변수로 받는 함수
   * **템플릿명 == 1개의 자료형명**

2) 작성법
   * **templete <class '템플릿명'>**
     **'자료형명' '함수명' ('템플릿명', '템플릿명');**
```
templete <class Any>
float sum(Any, Any);	// 'Any' 자료형 매개변수 & float형 반환
```
