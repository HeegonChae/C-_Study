## 공용체(union) & 열거체(enum)
### 공용체(union)
1) 서로 다른 데이터 형을 한 번에 한 가지만 보관 가능 // 구조체와 비슷한 구조
2) 장점) 메모리 공간이 **가장 큰 데이터형 크기**의 1개 같은 메모리 공간만 **공유 할당하므로 절약**
3) 특징) 마지막에 저장된 값만 유효
	
### 열거체(enum)
1) 기호 상수(only 정수)를 만드는 것에 대한 또 다른 방법 e.g. red == 1
2) 딕셔너리와 유사한다 생각
3) 열거자끼리 산술 연산 X **단, int형 변수와의 연산은 가능**
4) 빈 선언시 자동으로 채움(0)
